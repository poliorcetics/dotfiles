# Dotfiles

My dotfiles.

The configurations here may not work on your computer as no real effort has been made to make them portable beyond my needs.

## Available

- `tmux`: two themes and and config file.
- `vim`: the `peaksea` theme (not mine) in its most recent version, my basic config (see `vim/rcs/base.vim`) and the plugins I use (see `vim/rcs/plugins.vim`). The package manager is `vim-plug`.
- `sh`: shell configuration, supporting only `bash` at the moment. `zsh` support is planned but not available yet.
